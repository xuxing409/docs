## 主进程与渲染进程通信
因为主进程和渲染进程分担不同的工作，因此项目中遇到主进程和渲染进程之间进行一些操作是很常见的

### 渲染进程
```
import { ipcRenderer } from "electron";

// 监听渲染进程方法
ipcRenderer.send('window-close')
```
### 主进程
```
import { app, BrowserWindow, ipcMain } from "electron";

// 监听渲染进程方法
ipcMain.on("window-close", (e, data) => {
 win.close()
});
```
